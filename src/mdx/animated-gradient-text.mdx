# Animated Gradient Text Component

A text component that displays text with beautiful, animated gradient colors.

- Smooth animated gradient effects
- Customizable color schemes
- Adjustable animation speed
- Works with any text element (headings, paragraphs, etc.)

---

## Installation

The AnimatedGradientText component is part of our UI library. You can import it directly from the components directory.

<AnimatedGradientTextInstallation/>

---

## Usage

Here is how to use the animated gradient text in your project.

```tsx
import { AnimatedGradientText } from "@/components/modern-ui/animated-gradient-text"
```

<br/>

```tsx
<AnimatedGradientText>Beautiful gradient text</AnimatedGradientText>
```

## API Reference

### AnimatedGradientText

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `as` | `ElementType` | `'p'` | The HTML element to render |
| `colors` | `string[]` | `['#FF4D4D', '#F9CB28', '#9B4DFF', '#0070F3']` | Array of color strings for the gradient |
| `animationDuration` | `number` | `5` | Duration of the animation in seconds |
| `className` | `string` | - | Additional CSS classes |
| `children` | `ReactNode` | - | The text content |
